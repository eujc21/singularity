# GroupBy

Example :

```js
Collections.Projects()
  .where('serviceType', 'in', [ 'HVAC', 'Plumbing' ])
  .groupBy('serviceType', ($group) => {
     $group.sum('cost', 'totalSpend')
           .avg('cost', 'avgSpend')
           .min('cost', 'minSpend')
           .max('cost', 'maxSpend')
           .first('createdAt', 'firstCreated')
           .last('createdAt', 'lastCreated')
           .first('clientId')
           .build();
  })
  .aggregate()
  .then((result) => { console.log(result) })
```

MongoDB Aggregation :

```js
[
  {
    "$group": {
      "totalSpend": {
        "$sum": "$cost"
      },
      "avgSpend": {
        "$avg": "$cost"
      },
      "minSpend": {
        "$min": "$cost"
      },
      "maxSpend": {
        "$max": "$cost"
      },
      "firstCreated": {
        "$first": "$createdAt"
      },
      "lastCreated": {
        "$last": "$createdAt"
      },
      "clientId": {
        "$first": "$clientId"
      },
      "_id": "$serviceType"
    }
  },
  {
    "$match": {
      "serviceType": {
        "$in": [
          "HVAC",
          "Plumbing"
        ]
      }
    }
  }
]
```
